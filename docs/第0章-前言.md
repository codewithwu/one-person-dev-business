# 前言

我叫 copper，一个写了很多年代码、折腾了很多年副业的程序员。

几年前我整理过一份程序员挣外快的经验帖，发在网上，陆陆续续收到不少反馈和讨论。后来每次想更新，都觉得一篇帖子装不下了，但又一直没动笔。

六年过去了。这六年里我经历了几件事：众筹做课、尝试独立开发、做自媒体、从兼职副业过渡到全职自由职业。踩了不少坑，也积累了一些真正有用的认知。

但真正让我觉得「必须重写」的，是 AI。

2024 年我开始重度使用 AI 编程工具。一开始只是觉得写代码快了点，后来发现不是「快了点」——是整个游戏规则变了。一个人能做到的事情，边界被大幅推远了。以前需要团队的事，现在一个人加几个 AI 就能干。以前不敢想的产品方向，现在可以试了。

这本书不是原版的修订版。原版讲的是「程序员挣零花钱的渠道清单」，这本讲的是「2026 年，一个程序员怎么靠自己建一家公司」。

目标也不同了。不再是「挣点零花钱」，而是：**体面地、可持续地，靠自己的技能和 AI 获得独立收入。**

## 谁该读这本书

- 白天上班、晚上想搞点事的程序员
- 已经在做副业但收入卡在瓶颈的人
- 想从公司出来单干但不知道怎么起步的技术人
- 对「程序员 + AI」组合感兴趣、想看看能做什么的开发者

## 谁不该读这本书

- 想找暴富方法的人（这里没有）
- 想用大量资金创业的人（这不是创业指南）
- 想学投资理财的人（股票基金加密货币不归这本书管）

## 怎么读这本书

第一部分讲认知——为什么现在是最好的时机，怎么想清楚资源分配的问题，AI 到底改变了什么。

第二部分讲路径——四条经过验证的挣钱路线，每条都有具体的平台、工具和操作方法。

第三部分讲实战——从零到一把产品做出来、卖出去、持续赚钱的完整流程。

第四部分讲进阶——避坑、心态、从副业到一人公司的转变。

你可以从头读到尾，也可以直接跳到你最关心的章节。每章相对独立，但前后有逻辑关联。
