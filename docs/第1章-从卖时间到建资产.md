# 一个人就是一支队伍

## 一个真实的故事

2024 年底，我用 Claude Code 写了一个小工具——一个把 Markdown 笔记自动同步到多个平台的 CLI。从构思到发布 v1.0，用了四天。

四天。包括写代码、写文档、做 logo、发布到 npm、写了一篇介绍文章。

如果回到 2020 年，同样的东西我估计要花三周。不是因为我变厉害了，而是工具变了。AI 帮我写了 60% 的代码，帮我生成了 logo 的初稿，帮我把文档从中文翻译成英文，帮我写了 npm 包的发布配置。

这个工具后来每个月带来几百美元的被动收入。不多，但它证明了一件事：**一个人在 2026 年能做到的事，比五年前多太多了。**

## 到底变了什么

表面上看，变化的是工具。Cursor、Claude Code、Copilot 让写代码变快了。v0 让做 UI 变简单了。AI 翻译让出海变容易了。

但底层的变化更大：

**一个人的产出上限被大幅抬高了。** 以前一个全栈工程师一天能写 200 行有效代码，现在借助 AI 可以写 800-1000 行。不是粗制滥造的代码——AI 写初稿、你审查修改，质量是有保证的。

**技能的短板被补上了。** 很多程序员不擅长设计、不擅长写文案、不擅长做营销。以前这些是硬伤，现在 AI 可以帮你做 70 分的初稿，你只需要调整到 80 分。

**试错成本降低了。** 做一个 MVP 验证想法，以前要一个月，现在一周。这意味着同样的时间你可以试更多方向，找到可行方案的概率大大增加。

## 基础设施也成熟了

不只是 AI。整个独立开发的基础设施在 2026 年已经非常完善。

**收款**：Stripe、LemonSqueezy 一天之内接通全球支付。国内有微信支付个人商户。

**部署**：Vercel、Railway、Fly.io 一键部署，不用管服务器。

**数据库**：Supabase、Neon 免费层级够用。

**认证**：有成熟的开箱即用方案，不用自己从头写。

**分析**：PostHog、Plausible 自托管免费。

以前搭建这套东西要一个后端工程师忙活一周，现在半小时搞定。

## 机会在转移

AI 不只是降低了做的门槛，它同时在重塑市场格局。

**新机会出现了。** 围绕大模型的周边工具、MCP 服务器、AI 工作流编排、垂直行业的 AI 落地——这些是五年前不存在的市场。

**旧机会在消失。** 简单的 CRUD 外包、入门级编程教程、低门槛的模板市场——这些正在被 AI 蚕食。

**机会在向两端移动：** 一端是「高杠杆」——做一个 AI 工具撬动大量用户；另一端是「深垂直」——在一个细分行业做到比通用 AI 更懂行。

中间地带（通用技能、通用产品）的竞争会越来越激烈。

## 一人公司不是创业

需要说清楚一件事：一人公司和创业是两回事。

创业追求增长——融资、扩张、做大、上市或卖掉。

一人公司追求的是另一种东西：**用最小的团队（就你一个人），获得足够的收入，过自己想过的生活。**

不融资。不招人。不追求无限增长。利润率优先于营收规模。自由度优先于公司估值。

这不是退而求其次。这是主动选择。
